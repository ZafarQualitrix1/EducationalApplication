# Vercel deployment instructions

1. Deploy only the frontend folder to Vercel.
2. In Vercel dashboard, set project root to frontend.
3. Build command: npm run build
4. Output directory: build
5. vercel.json is already present in frontend.
6. Backend should be deployed to Railway, Heroku, or similar.

If you want to deploy backend on Vercel, use Next.js API routes or serverless functions.

For any errors, run npm run build in frontend locally and fix issues before deploying.
